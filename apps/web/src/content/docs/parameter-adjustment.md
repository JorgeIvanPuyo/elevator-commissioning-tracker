# Ajuste de parámetros

## Parámetros manuales

Parámetros usados para ajuste manual y comportamiento fino:

- 212
- 214
- 022F
- 229

## Parámetros de bias

Los parámetros 026D en adelante se capturan como valores HEX con conversión decimal.

## Warnings min/max

Las inconsistencias min/max no bloquean el guardado. Se muestran como advertencias para mantener trazabilidad del valor entregado o capturado.
